React Flickr Viewer
====================

A Viewer for viewing Flickr public feed using react.

To build run:

    $ npm install
    $ npm run bundle

The application will build into the dist folder.

To run the tests run:
		
    $ npm test

Or to calculate code coverage run:
    
    $ npm run coverage
